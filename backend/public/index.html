Usuarios

GET
/api/users

Obtiene todos los usuarios

POST

/api/users

Crea un nuevo usuario

POST

/api/users/login

Inicia sesión con un usuario

GET

/api/users/:id

Obtiene un usuario por ID

DELETE

/api/users/:id

Elimina un usuario por ID

PUT

/api/users/:id

Actualiza un usuario por ID

Posteos

GET
/api/posts

Obtiene todos los posteos

GET
/api/posts?sort=top

Obtiene todos los posteos ordenados de mayor a menor likes

GET
/api/posts?sort=views

Obtiene todos los posteos ordenados de mayor a menor vistas

POST

/api/posts

Crea un nuevo posteo

GET

/api/posts/user/:userId

Obtiene un posteo de un usuario por su ID

GET

/api/posts/:id

Obtiene un posteo por ID

GET

/api/posts/name/:title

Obtiene un posteo por su titulo

DELETE

/api/posts/:id

Elimina un posteo por ID

PUT

/api/posts/:id

Actualiza un posteo por ID
